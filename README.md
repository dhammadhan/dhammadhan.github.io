# Dhamma Dhan

Dhamma Dhan is the practice of building wealth in alignment with right action. This site showcases principles, practices, and resources for ethical wealth building.

## About

This repository hosts the personal website and blog for Dhamma Dhan, exploring the intersection of Buddhist principles and financial well-being.

## Features

- Clean, minimal design
- Blog posts on ethical wealth practices
- Responsive layout for all devices
- Built with Jekyll and hosted on GitHub Pages

## Getting Started

### Prerequisites

- Ruby 2.7.0 or higher
- Bundler

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/dhammadhan/dhammadhan.github.io.git
   cd dhammadhan.github.io
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Run the local server:
   ```bash
   bundle exec jekyll serve
   ```

4. Visit `http://localhost:4000` in your browser.

## Structure

```
├── _posts/          # Blog post markdown files
├── _layouts/        # HTML layout templates
├── _includes/       # Reusable HTML components
├── assets/          # CSS, JavaScript, and images
├── _config.yml      # Jekyll configuration
└── index.html       # Homepage
```

## Writing Posts

Create a new markdown file in the `_posts` directory with the filename format:
```
YYYY-MM-DD-post-title.md
```

Include frontmatter at the top:
```yaml
---
layout: post
title: "Your Post Title"
date: 2026-06-17
categories: [wealth, ethics]
---
```
